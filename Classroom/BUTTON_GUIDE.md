### **Engagement Button** (ğŸ“Š)
![Engagement Button Icon]

**Location:** Top toolbar, right side

#### What It Does
Opens the **Engagement Dashboard** - a real-time analytics panel that shows you how each student is performing and participating in the session.

#### What You Can See
- **Engagement Score** (0-100%) for each student
- **Attention Level** (Focused / Distracted / Away / Inactive)
- **Comprehension Estimate** (AI-derived understanding score)
- **Participation Count** (recent interactions)
- **Struggle Indicators** (AI-detected confusion signals)
- **Time in Session** for each student

#### Available Actions Per Student
| Action | Description |
|--------|-------------|
| Send Nudge | Send an encouraging message to a struggling student |
| 1:1 Session | Invite student to a private breakout room |
| View Details | See detailed student analytics |

#### Filters Available
- **All** - Show all students
- **Struggling** - Students with low engagement or comprehension
- **Inactive** - Students who appear away or idle
- **Engaged** - High-performing students

#### How to Use
1. Click the "Engagement" button to open the panel
2. View the list of students with their metrics
3. Click on a student to expand detailed view
4. Use quick action buttons to intervene
5. Use filters to focus on specific groups

#### When to Use
- At the start of class to check who's engaged
- When you notice participation dropping
- To identify students who might need help
- To send targeted encouragement

### **Polls Button** (ğŸ“Š)
![Polls Button Icon]

**Location:** Top toolbar, right side  
**Badge:** Shows number of active polls

#### What It Does
Opens the **Quick Polls Panel** - a tool to create interactive polls and quizzes for real-time student feedback.

#### Poll Types Available
| Type | Use Case |
|------|----------|
| Multiple Choice | Check understanding of concepts |
| True/False | Quick comprehension checks |
| Rating (1-5) | Gauge confidence levels |
| Short Answer | Open-ended questions |
| Word Cloud | Collective brainstorming |

#### Built-in Templates
1. **"Do you understand this concept?"** (Yes/No)
2. **"How confident are you?"** (1-5 rating)
3. **"Which topic should we review?"** (Multiple choice)
4. **"Ready to move on?"** (Yes/No)

#### Creating a Poll
1. Click the "Polls" button
2. Choose a template or click "New Poll"
3. Enter your question and options
4. Set poll settings (anonymous, time limit, etc.)
5. Click "Create Poll"
6. Click "Start Poll" when ready

#### During a Poll
- Students see the poll and can respond
- Live results update in real-time
- Participation rate is displayed
- You can see which students have voted

#### After a Poll
- Click "End Poll" to close it
- View final results with statistics
- Results can be shown to students or kept private
- You can run the same poll again if needed

#### When to Use
- To check understanding before moving on
- To gather opinions on topic choices
- To make the class interactive
- To identify areas that need more explanation

## ğŸ¯ Quick Tips

### For Engagement Dashboard
- Check it at the beginning of class to see initial engagement
- Watch for red/yellow indicators showing struggling students
- Use the "Struggling" filter to focus on students needing help
- Send nudges sparingly - too many can be distracting

### For Polls
- Use polls every 10-15 minutes to maintain engagement
- Keep poll questions short and clear
- Anonymous polls get more honest responses
- Review results before moving to the next topic

### For Command Palette
- Memorize key shortcuts: âŒ˜M (mute), âŒ˜V (video), âŒ˜S (screen)
- Use it when you don't want to look away from students
- Type any keyword to find actions quickly

### For Whiteboard
- Use different colors to organize information
- The eraser has a larger brush than the pen
- Text can be moved by clicking and dragging after placement
- Download the whiteboard at the end of class for records

## â“ Troubleshooting

### Engagement button doesn't show data
- Make sure students are connected to the session
- Wait 30 seconds for initial data to load
- Check that socket connection is active (green dot in header)

### Polls button shows no active polls
- Click the button to open the panel
- Create a new poll using "New Poll" button
- Or select a template to get started quickly

### Command palette doesn't open
- Try clicking the Command button instead of keyboard shortcut
- Make sure you're focused on the browser window
- Check browser console for JavaScript errors

### Whiteboard tools not working
- Make sure you're not in "read only" mode
- Try refreshing the page if canvas seems frozen
- Check browser compatibility (Chrome/Firefox recommended)
